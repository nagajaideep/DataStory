# Data Story —— Emotion & Sentiment Benchmarks

A comparative analysis of the best open-source models for understanding human emotions.

### Models Featured
- **DistilRoBERTa**: Lightweight and fast.
- **GoEmotions**: Granular 28-label taxonomy.
- **BART-MNLI**: Zero-shot flexibility.
- **Twitter-RoBERTa**: SOTA sentiment (3-class).

### Key Features
- Plutchik's 8-emotion framework grid.
- Custom model cards with F1 score stats.
- Comparative accuracy tables.
